*Phía html
1. Bôi đen điểm giống nhau của các kí tự trước id cần copy
2. Nhấn Ctrl + Shift + L
3. Nhấn mũi tên sang phải
4. Nhấn Ctrl + Shift + Mũi tên sang phải
5. Nhấn Ctrl + C để copy
*Phía css
1. Ctrl + V để paste
2. Giữ con trỏ ở đầu tên id đầu tiên
3. Giữ ALT và chọn đầu tên các id tiếp theo
4. Thêm các kí tự cần thêm
5. Ấn Ctrl + Mũi tên phải để di chuyển các con trỏ chuột sang phải
6. Thêm các kí tự cần thêm
